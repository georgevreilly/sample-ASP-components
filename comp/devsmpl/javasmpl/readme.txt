Overview
========

This is the Java version of the Simple sample object for Active Server
Pages.  It demonstrates how to write a component that has
straightforward methods and properties.

Please read the top-level ReadMe for all the Simple and Power samples.

Look at JavaSmpl.java to see how these methods and properties are
declared and implemented.


File List
=========

File             Description
----             -----------

.\Source

 JavaSmpl.java   the source code for the Java simple component
 Makefile        a makefile that can be used with nmake
 JavaSmpl.mak    the Developer Studio makefile
 JavaSmpl.dsp    the Developer Studio project file
 reg.bat	 installs and registers the compiled .class file

.\Samples

 JavaSmpl.asp    Demonstrates the use of the Java Simple Sample object
